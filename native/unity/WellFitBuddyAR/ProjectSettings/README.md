# ProjectSettings Hinweise

Diese Datei ist eine Vorbereitung fuer das lokale Unity-Projekt.

Nach dem Erzeugen des Projekts speichert Unity echte ProjectSettings-Dateien in diesem Ordner.

Wichtige Einstellungen fuer WellFitBuddyAR:

- Unity Version: 2022.3 LTS als Startversion
- Build Target: Android zuerst
- XR Plugin Management aktiv
- Android XR Provider: ARCore aktiv
- iOS XR Provider: ARKit spaeter aktiv
- Camera Permission pruefen
- Orientation: Portrait oder Auto-Rotation spaeter entscheiden
- Development Build fuer erste Tests aktivieren

Nicht vergessen:

- Unity Library nicht committen
- Build-Ordner nicht committen
- Nur relevante Settings, Assets und Scripts versionieren
