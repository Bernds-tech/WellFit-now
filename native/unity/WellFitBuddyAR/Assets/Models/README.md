# Models

Hier kommen spaeter 3D-Modelle fuer Buddy, Items und AR-Marker hinein.

Empfohlen fuer den Start:

```txt
Models/Placeholder/
  BuddyPlaceholder.glb oder einfache Unity-Primitive
```

Zielwerte fuer mobile AR:

- geringe Polygonzahl
- kleine Texturen
- einzelne Basisassets idealerweise unter 2 bis 5 MB
- spaeter LODs pruefen
- keine schweren Shader im ersten Prototyp

Buddy-Typen spaeter:

- kleiner Drache
- Tier-Buddy
- Ritter/Zwerg/Fantasy-Buddy
- saisonale Varianten
