# Materials

Geplante Materialien:

```txt
BuddyPlaceholder.mat
ShadowBlob.mat
HintMarker.mat
SurfaceDebug.mat
```

## BuddyPlaceholder

Einfaches warmes Buddy-Material fuer den ersten Prototyp.

## ShadowBlob

Ein einfacher transparenter Bodenschatten, damit der Buddy glaubwuerdiger auf Flaechen steht.

## HintMarker

Material fuer AR-Hinweise und Marker.

## SurfaceDebug

Material fuer erkannte Flaechen im Development Build.

Hinweis: Spaeter mobile-optimiert halten und keine schweren Shader im ersten ARCore-Test nutzen.
