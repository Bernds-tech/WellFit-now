# Prefabs

Geplante Prefabs:

```txt
BuddyPlaceholder.prefab
SurfaceNode.prefab
HintMarker.prefab
ShadowBlob.prefab
```

## BuddyPlaceholder

Erster einfacher Buddy, zum Beispiel Capsule, Cube oder Platzhalter-Drache.

Komponenten:

- BuddyController
- BuddyLookAtCamera
- BuddyNavigationController
- BuddyAbilityController

## SurfaceNode

Repraesentiert erkannte oder markierte Flaechen.

Komponenten:

- BuddySurfaceNode

## HintMarker

Markiert Hinweise, Objekte oder Orte in AR.

Komponenten:

- ArMissionHintMarker
