## Build-/Deployment-Hinweis

Mehrere Web-/TypeScript-Aenderungen liegen auf `main`, muessen aber fuer `wellfit-now.io` noch per Server-Build/Deploy uebernommen werden.

Root-Pruefrahmen bleibt:

```bash
cd /var/www/WellFit-now
git fetch origin
git reset --hard origin/main
npm install --include=dev --no-audit --no-fund
NODE_OPTIONS="--max-old-space-size=768" npm run build
pm2 restart wellfit-now --update-env
pm2 status
pm2 logs wellfit-now --lines 30
```
