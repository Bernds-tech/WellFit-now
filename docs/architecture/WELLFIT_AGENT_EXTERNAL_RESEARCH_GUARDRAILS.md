# WELLFIT AGENT EXTERNAL RESEARCH GUARDRAILS

Status: planning_only (report-only research)

## 1) Was externe Recherche darf
- Trends analysieren.
- Studien zusammenfassen.
- Wettbewerber vergleichen.
- UX-/Motivationsmuster auswerten.
- Vorschlaege machen.

## 2) Was externe Recherche nicht darf
- Keine ungeprueften Quellen als Wahrheit uebernehmen.
- Keine medizinischen Diagnosen.
- Keine Kinder-/Health-/Location-Mechanik ohne Review.
- Keine Dark Patterns.
- Keine suechtig machende/manipulative Optimierung als Ziel.
- Keine Token/Payment/Cashout/Blockchain-Aktivierung.

## 3) Quellenqualitaet
- Bevorzugt wissenschaftliche Studien, offizielle Guidelines, serioese Branchenberichte.
- Quellen muessen zitiert werden.
- Veraltete Informationen markieren.
- Bei Widerspruch mehrere Quellen nennen.

## 4) Behavioral Design Leitlinien
- Autonomie, Kompetenz, soziale Verbundenheit.
- Spass statt Zwang.
- Fortschritt statt Scham.
- Team/Community ohne Gruppendruck.
- Personalisierung ohne sensible Rohdaten.
- Keine Casino-/Jackpot-/Gambling-Sprache.
